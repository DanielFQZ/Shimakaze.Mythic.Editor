# Shimakaze.Mythic.Editor
适用于Minecraft服务器插件MythicMobs[神话怪物]的编辑器  
为mobs、items、spawners等文件的配置提供了一个对新手比较友好的图形界面  
因为是第一个版本，所以还有很多地方需要完善
